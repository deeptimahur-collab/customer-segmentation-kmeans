# 🧠 Customer Segmentation using K-Means Clustering

## 📌 Project Overview
This project applies K-Means clustering to segment customers based on their purchasing behavior. The goal is to identify distinct customer groups that can help businesses make data-driven marketing decisions.

---

## 🎯 Objective
- Analyze customer data to uncover hidden patterns
- Group customers into meaningful segments
- Provide actionable insights for targeted marketing strategies

---

## 🛠️ Tech Stack
- Python 🐍
- Pandas
- NumPy
- Matplotlib / Seaborn
- Scikit-learn

---

## 📊 Dataset
- Mall Customer Dataset
- Features used:
  - Age
  - Annual Income
  - Spending Score

---

## ⚙️ Steps Performed
1. Data Cleaning and Preprocessing  
2. Exploratory Data Analysis (EDA)  
3. Feature Selection  
4. Elbow Method to determine optimal clusters  
5. K-Means Clustering  
6. Visualization of clusters  

---

## 📈 Results & Insights
- Identified **4–5 distinct customer segments**
- High-income & high-spending customers → Premium target group  
- Low-income & low-spending customers → Low engagement segment  
- Mid-income groups → Potential growth segment  

💡 These insights can help businesses:
- Improve targeted marketing  
- Increase customer retention  
- Optimize product offerings
